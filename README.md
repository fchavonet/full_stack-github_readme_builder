## ğŸ“„ <span id="description">Description</span>

GitHub README Builder is an interactive web application that allows users to compose, preview, and export GitHub README.md files. It offers a live Markdown editor, predefined content modules, custom badge generation, and instant preview styled with GitHub-like formatting.

## ğŸ“ <span id="objectives">Objectives</span>

- Provide a user-friendly Markdown editor focused on GitHub READMEs.
- Enable quick insertion of structured sections (titles, objectives, tech stack...).
- Render live previews with GitHub-like styling.
- Offer badge generation using [Shields.io](https://shields.io/) and [Simple Icons](https://simpleicons.org/).
- Improve DOM manipulation and UI/UX handling using vanilla JavaScript.

## ğŸ’» <span id="installation_and_how_to_use">Installation and how to use</span>

**Installation:**

1. Clone this repository:
    - Open your preferred Terminal.
    - Navigate to the directory where you want to clone the repository.
    - Run the following command:

```bash
git clone https://github.com/fchavonet/web-github_readme_builder.git
```

2. Open the repository you've just cloned.

**How to use:**

1. Open the `index.html` file in your web browser.

2. Start writing your Markdown content in the editor area.

3. Use the sidebar buttons to insert predefined modules (title, objectives, badge...).

4. Click copy or download button to save your README file.

You can also test the web application online by clicking [here](https://fchavonet.github.io/web-github_readme_builder/).
